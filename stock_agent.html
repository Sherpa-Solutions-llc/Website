Stock Monitor Pro Documentation

Overview

Stock Monitor Pro is a real-time algorithmic market scanning and trend analysis engine. It allows you to monitor thousands of equities simultaneously against customizable volume and price-action criteria.

Dashboard Features

  • Real-Time Scanning (Scan Control Panel): The primary engine. Clicking 'Start Scan' initiates a simulated backend market sweep lasting approximately 45 seconds. The engine filters thousands of US Equities against the parameters defined in your Criteria Search tab, returning visually prioritized matches (bullish vs bearish signals).
  • Criteria Search (Settings): Defines the constraints for the Real-Time Scan. You can customize Minimum Daily Volume (e.g., Over 2M), Minimum Relative Volume (e.g., >3x average), and establish an absolute Maximum Share Price ceiling to target specific market caps.
  • Stocks to Watch (Watchlist): Your persistent, cloud-synchronized database of target assets. You can populate this list either by clicking the 'Watchlist' button next to a successful scan result, or by manually typing a ticker symbol into the input field at the top of the tab. Stocks can be removed via the trash icon.
  • Strategies: The core logic algorithms that power the Analytics Engine. You can toggle one or multiple trading algorithms ON simultaneously (e.g., Momentum Breakout, Gap and Go). Clicking a toggle opens a configuration modal allowing you to fine-tune capital allocation, stop-loss triggers, and take-profit targets specific to that strategy.
  • Analytics Engine: The automated cross-referencing system. You can set the engine to "Auto-Evaluate" at customizable frequencies (1 Minute up to 1 Hour) via the dropdown. When active, it iteratively tests every stock in your Watchlist against every Strategy you have toggled ON. If a stock meets the criteria of a strategy, it generates a theoretical 'Confidence Score', flashes the Analytics tab blue, triggers a sonic alert, and populates the table with an actionable 'Execute Trade' button.
  • App Logs: A real-time terminal window outputting diagnostic information, system status changes, scan progress nodes, and database sync verifications.
  • Settings & Broker Integrations: A secure panel to input custom API keys. 'Data Source Integrations' allow you to bypass the deterministic mock engine by supplying an Alpha Vantage developer key. 'Brokerage Integration' allows you to theoretically link a live execution or paper trading account (Alpaca, Interactive Brokers, TD Ameritrade, Tradier) using an API Key and Secret, which authorizes the Analytics Engine 'Execute Trade' buttons.
  • Global Header Controls:
    • Audio Toggle: Enables or disables the sonic 'ping' notification fired when the Analytics Engine finds a match. Default is ON.
    • Documentation: Opens this informational modal.
    • Exit: Returns the user to the main Sherpa Solutions Project Portfolio page.

Quantitative Analysis Engine & Infrastructure

High-Frequency Data Ingestion Pipeline

The system utilizes asynchronous Websocket streams and high-capacity REST polling against aggregated financial feeds to maintain millisecond-accurate order book states.

Algorithmic Evaluation Engine (FastAPI)

Built on Python FastAPI, the backend handles concurrent evaluation of hundreds of ticker streams against predefined technical indicator thresholds (RVOL, Price-to-Earnings anomalies, Volume surges).

Event-Driven Architecture

Instead of continuous frontend polling, the engine pushes state updates directly to the client via WebSockets, ensuring zero latency between algorithm triggers and UI updates.
